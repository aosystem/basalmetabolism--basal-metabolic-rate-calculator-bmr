# BMR Calculator 🏃‍♂️⚖️

A lightweight and precise Basal Metabolic Rate (BMR) calculator built with **Flutter**. This application provides an intuitive interface for quick health and fitness assessments.

## 🌟 Key Features
- **Accurate Calculation:** Uses standardized formulas to determine BMR.
- **Cross-Platform:** High-performance UI/UX powered by Flutter for both Android and iOS.
- **Privacy First:** All calculations are performed locally on the device. No personal health data is collected.
- **Open Source:** Core business logic is transparent and open for the community.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Architecture:** Clean and optimized logic for mobile devices.

*Note: This repository contains the core logic and source code. Assets (images, fonts, and localization files) are excluded to protect intellectual property.*
